# TC-044: Успешное получения статуса подписки при повторной оплате 3 месяцев


**Предусловия:**
1. Войти в приложение "Telegram"
2. Авторизоваться в приложении
3. Перейти в бот по ссылке: https://t.me/Argocanalbot
4. Нажать на кнопку "Старт"
5. Нажать на кнопку "Получить доступ"
6. Ввести в поле "Сообщение" - Анастасия
7. Нажать на кнопку отправки сообщения

**Шаги:**
1. Выполнить шаги из ТК 37
2. Нажать на кнопку "Статус подписки"

**Ожидаемый результат:**
Появляется сообщение "Доступ активен до (дата - 90 дней с первой оплаты и 90 дней с момента повторной оплаты)"

**Статус:** ✅ Pass
